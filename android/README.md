GoldFish Android App
====================

System Requirements
-------------------

* Android 2.3.1+
* NFC

Config
======

Set Base URL
------------

edit *base_url* in "android/res/values/string.xml".


Build and Install Android App
=============================

With Eclipse
------------

Import "android" directory into your eclipse workspace and build it.


With AndroidSDK
----------------

    % cd android
    % make setup
    % make
    % make install
